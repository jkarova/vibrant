Grading

Grading will be based on the following:


  • Homework 30%
  • In-class presentations 10%
  • Midterm exam 30%
  • Final Presentation 30%

Homework is due by the beginning of class on the posted date and must conform to the standards for delivery. Late assignments are not accepted, and homework is not accepted through email.

You must be present to give your final presentation to pass the course.


Standards for Delivery

Submit answers to all the exercises in an assignment through dlweb by the beginning class on the day the assignment is due.

For questions requiring written answers, submit a text file (not a Word document, or an rtf file - a plan text file) containing the answer to the question - including all parts to the question if there are multiple parts. For questions requiring the submission of a POV Ray file, submit the .pov file and all .inc files necessary to render the image. All submitted files must be named according to the following scheme:

     first initial + your last name + '_' + hw#q#

For example, if John Lasseter was submitting a text file as an answer to homework 4, question 3, his file prefix would be jlasseter_hw4q3.txt. If Edward Catmull was submitting a POVRay file as an answer to homework 2, question 2, his file prefix would be ecatmull_hw2q2.pov.

Submitted POVRay files must also contain a header with the following documentation:

  • Your name
  • The name of the file
  • The assignment and question numbers
  • A description of what the file does (brevity is key here)

Multiple files can be zipped together and submitted through dlweb. If you submit a zip file, name your file using your first initial, your last name, and the assignment number. For example, if John Lasseter was submitting assignment 2, his zip file would be named jlasseter_assignment2.zip.

The following gaffes will result in automatic point deductions:

  • A zip file that is unreadable or cannot be opened: 10 points first infraction, 25 points on each subsequent infraction. (Unzip your file and test it before uploading it.)
  • A naming scheme that does match the one outlined above: 5 pts. per file
  • Failure to include documentation: 5 pts. per problem
  • Failure to submit a zip file before class begins: 50 pts.

Note that it is possible to answer all the question in an assignment correctly and still fail the assignment.
